1. Navigate to the AFDCO CVA/H Document & Training SharePoint site
  2. Select the required version folder under Rapid Cyber Documentation/Guides
  3. Navigate to and open the required platform guide folder
  4. Right-click on the required HTML guide file and select download
  5. Navigate to the saved download location and select the file
  6. The default location is likely the download folder
  7. Add a Space Name
  8. Click on the Upload <document zip file> button
  9. Wait for the confirmation message
  10. Refresh the page
  11. Select the Space Name in the Navigation bar
