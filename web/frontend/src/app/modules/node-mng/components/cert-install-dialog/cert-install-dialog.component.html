Certificate Installation Instructions


Certificate Password

When importing the certificate, the password to use is: 'bdppassword1'.

Google Chrome Certificate Import

To import the certificate, first click the 'Download' button below to save the certificate to your local system. Then execute the following steps...

  • In Chrome, click the 3 vertical dots (ellipsis) button to bring up the menu
  • Select 'Settings'
  • On the left side, click 'Privacy and security'
  • On the right side under 'Privacy and security', click 'Security'
  • On the right side under 'Security', click 'Manage certificates'
  • In the popup window, click 'Import...'
  • In the popup window, click 'Next'
  • In the popup window, click 'Browse'
  • In the file dialog window, change the extensions to filter on to 'Personal Information Exchange (*.pfx,*.p12)'
  • In the file dialog window, select the downloaded certificate
  • In the file dialog window, click 'Open'
  • In the popup window, click 'Next'
  • In the popup window, type in the password from above
  • In the popup window, click 'Next'
  • In the popup window, click 'Next'
  • In the popup window, click 'Finish'
  • In the popup dialog, click 'OK'
  • In the popup window, click 'Close'
