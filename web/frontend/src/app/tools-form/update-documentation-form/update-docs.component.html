chevron_right expand_more Update Controller Documentation
  1. Navigate to the required DI2E confluence space.
  2. Use the PAGE TREE on the left menu to navigate to, and select the required page.
  3. Click on the ... button in the top right.
  4. Click on Export to Html (Note: For offline documentation you can also select Export to PDF. You will need HTML documentation if you wanted to update the controller.)
  5. A modal popup will appear make sure "Export Profile" is set to No Profile.
  6. Ensure that "Select Content" is set to Current Page with all child pages.
  7. Click Export button. The download will take some time to complete, so grab a cup of coffee.
  8. Click on the Choose Zip File button at the bottom of this box.
  9. Navigate to where you saved the downloaded documentation and select it. More than likley it will be in your downloads folder.
  10. Add a Space Name
  11. Click on the Upload <documentation zip file> button. Wait for confirmation message.
  12. Finally select the Space Name link in the navigation bar!
